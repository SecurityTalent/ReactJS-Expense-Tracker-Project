# Expense Tracker

This is a simple expense tracker built using ReactJS.

## Features

* User can add an expense with a title and amount
* User can delete an expense
* The total amount of all expenses is displayed

## How to use

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser

## Technologies Used

* ReactJS
* Vite
* ESLint
* Prettier

## Folder Structure

* `src` - contains the source code for the application
	+ `App.jsx` - contains the main application component
	+ `Component` - contains the components for the application
		- `ExpenseForm.jsx` - contains the form component for adding an expense
		- `ExpenseList.jsx` - contains the list component for displaying the expenses
		- `ExpenseItem.jsx` - contains the item component for displaying an expense
	+ `App.css` - contains the CSS styles for the application
* `vite.config.js` - contains the configuration for Vite
* `eslin.config.js` - contains the configuration for ESLint
* `package.json` - contains the dependencies and scripts for the application
